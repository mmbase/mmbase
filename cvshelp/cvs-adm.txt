CVSROOT files

checkout met cvs checkout CVSROOT

Files:
config
system parameters
SystemAuth=no
LockDir dir voor lockfiles normaal niet gebruikt.
/usr/local/cvslocks bv.

modules
alternatieve namen voor subdirs (subprojecten)
mmbaseroot nl/org/mmbase/core 

commitInfo rcsInfo logInfo
hooks om programma's te runnen op een commit e.d.
format : regexp program-to-run
regexp DEFAULT or ALL doen het altijd
logInfo reageert op de logmessage (voor commit emails b.v.)
parameters voor program %s (naam file) %V revisie voor commit %v revisie na commit
%{sVv} geeft comma gescheiden groepen

cvswrappers
definietie hoe te reageren op bepaalde files
*.jpg -k 'b' b.v.
*.class -k 'b'
*.gif -k 'b'

Keyword expansion

$Author: wwwtech $ laatste gecomitte username
$Date: 2000-01-24 17:12:56 $ date of laatste commit
$Revision: 1.1.1.1 $ revisie
$Id: cvs-adm.txt,v 1.1.1.1 2000-01-24 17:12:56 wwwtech Exp $ id string met author,date,revisie en file
$Source: /space2/mmbase.org/cvshelp/cvs-adm.txt,v $ path naar repository file
$Log: not supported by cvs2svn $ re-expanding log berichten (blijft groeien dus)

export CVS_CLIENT_LOG=file
voor debuggen
