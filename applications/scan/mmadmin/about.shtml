MMRunner version

Welcome to Runner the quick unzip & go version of the MMBase Content management system.

This is a full working MMBase setup that comes with several simple applications auto-deployed. Its goal is allow people to use MMBase within minutes of downloading the system and see what its all about. Its also handy to test applications without using your production machines/setups. This version is for people who are oke with the bundled server and database for the ohther setups download the source version of MMBase.

The MMbase Team.

License info & Thanks to

MMBase itself is released as OpenSource product (see main website). But to allow us to create this MMRunner version we needed to include a application server and a database. Since we wanted a simple and 100% java solution we used orion and hypersonic but MMBase itself supports many more both commercial and opensource products.

About Orion :

Orion is a great j2ee application server that is 100% java based, easy to install and follows all the new and hip specs. Its not free but at $1500 its very cheap indeed and it is free for development and non-profit use. Many thanks to the orion people for allowing us to include it in the demo version. More info on orion at www.orionserver.com.

About Hypersonic :

Hypersonic is a small 100% java based JDBC database, its opensource, fast very small (about 100kb). It also supports a memory only mode what we use in the demo meaning that its empty once you stop the server. For more info on Hypersonic at http://hsql.oron.ch.
