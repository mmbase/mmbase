Authentication

The Authentication class will validate a login, and on succes return a new UserContext object, which can be used to tell the authorization who we are. The authentication is done by providing a HashMap and the way how we want to login to our system. The authentication is retrieved from the MMBaseCop and depends on the setting in the security.xml which class it is. When the authentication is loaded, a filename is given to the authentication which can be used to get settings from a config file for the authentication. The authentication also contains a method isValid(), which checks if a userContext is still valid.
