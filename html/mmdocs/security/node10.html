basic

This implementation can be used, when the basic authentication/authorization is used by the system. For the basic implementation the following classes should be loaded.
  • org.mmbase.security.implementation.basic.AuthenticationHandler This class will load several modules, which can be used for authentication. The config file for this authentication can be found in the config directory security/basic/authentication.xml. In this xml the account.properties file can be specified, which then can be used to login.

  • org.mmbase.security.implementation.basic.OwnerAuthorization This class will check the authorization of the nodes which are retrieved. The checking is done by verifying the name of the owner field with the login name.
In the situation you want to have only the check to see if a user has logged in, but not the verification on the nodes, the class to use for authorization would be: org.mmbase.security.NoAuthorization. The authentication then will be active, but there will be no checks on the operations on the nodes itselve.
