Autorization

The Authorization class checks if a UserContext has the rights to perform a certain operation on a certain node. The Operations which can be done are the following:
  1. Read : If a node may be accessed.
  2. Write : If a node may be changed.
  3. Create : If a builder may create a new node.
  4. Link : If a relation may be created to the node.
  5. Delete: If the node may be deleted.
  6. Change Context: If the node rights may be changed.
This operations can be used inside the method: boolean check(UserContext, NodeId, Operation) which returns true is an operation is permitted, and the method void assert(UserContext, NodeId, Operation) which trows an exception when a operation is not permitted. Furthermore this class contains a few methods to notify the security system of changes in the system.
