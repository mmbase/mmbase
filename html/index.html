Congratulations with MMBase


MMBase is an opensource publishing system. MMBase can be used to create big websites that can be maintained easily.  MMBase separates content from layout, in this way information can be reused easily. MMBase is very unique in the way it maintains its information. MMBase creates an object cloud that is a digital representation of the 'real' world objects that are of interest for you.
 

The Admin tool can be found at (username=admin password=admin2k):

http://yourhost/mmadmin/index.shtml
 
The Editors can be found at (username=admin password=admin2k):
http://yourhost/mmeditors/index.shtml
 
The MMBase site can be found at:
http://www.mmbase.org
 
You can change the admin password by editing /org/mmbase/config/default/account.properties.
