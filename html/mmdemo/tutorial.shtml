MMBase step by step toturial

This online tutorial is still in its starting phase. The idea is to explain using working examples how mmbase works. For this reason we have created a application around forture cookies. Each example will be explained in one or more of the scripting langauges mmbase supports. The tutoral moves from retrieving things from a cloud, searching to creating, changing and removing things. In the last parts we will bring these together into a few example applications and refer to other demo's included in mmdemo/mmbase.
