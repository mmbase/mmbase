MMBaseCop

The class which provides the handle to our security system. There is an instance in the module MMBase, and this one can be used to obtain the authentication and autorization module. The MMBaseCop will load the MMBaseCopConfig, which is the configuration, which should be used by the MMBaseCop. The instance of the authenticator can be obtained by the the method: getAuthentication() and a instance of autorization can be obtained by the method: getAuthorization()
