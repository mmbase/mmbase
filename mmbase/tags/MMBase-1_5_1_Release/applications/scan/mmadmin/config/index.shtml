MMConfig
The Config module gives information on MMBase's configuration files. These can be:
  • Listed by category:
    • A green box indicates that the object associated with a configuration file is active, red stands for inactive.
    • If the file contains an xml error, this is indicated.
    • Link under the filename goes to tabular view of contents produced by XSL conversion
    • Link under xml goes to syntax colored display if the xml format checks allright and to a displayed annotated with error messages if not.
  • Tabular view of data in XML file generated by an XSL conversion
  • Displayed with syntax coloration (This is implemented on the basis of a successfull parse action)
  • For xml files with errors there is a display of the file with an indication of what error occurs where. (Xerces error notification isn't quite what it should be!)
