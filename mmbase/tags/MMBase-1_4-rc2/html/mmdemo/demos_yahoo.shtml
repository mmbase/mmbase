Description of MyYahoo

MyYahoo is a simple example of how to build a 'yahoo' website. This is a website where urls are categorised. This webpage will describe the design of this example. In /mmexamples/myyahoo/scan/ you can find the templates that create the actual site.

Cloud Design Picture

The image on the right side shows the design of this MyYahoo example.
MyYahoo consists of 3 object types, namely: jumpers, pools and urls. The jumper makes it possible to use a short url that will be expanded to a larger one. e.g. http://yourhost/myyahoo will be expanded to http://yourhost/mmexamples/myyahoo/scan/index.shtml?yahoo.
The pools are used to categorise urls. On pool contains all urls that belong to the categorie music. The other pool contains all urls that belong to the categorie movies. One pool is used as an exception, it contains all pools that function as a categorie.


Manual Picture

The image at the right side shows the layout of the MyYahoo example. The url at the top of this page allows you to see the real MyYahoo example.
MyYahoo shows some categories for the urls, namely: musix, movies, partners. At the bottom all urls existing in your system are displayed. The top of the page gives you the possibility to search for urls. Try the example and look at the templates! It will give you a good view about how MMBase works.
