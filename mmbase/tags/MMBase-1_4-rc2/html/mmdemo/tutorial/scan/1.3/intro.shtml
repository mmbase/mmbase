1.3 Listing mode than 1 node
Ofcourse its important not only to get content from just one node that you allready know the number of but you also want to list / search them based on their type. In this example we list all the nodes of the url type.

We use the 'LIST OBJECTS' command for this, this command has 3 parts, the 'LISTOBJECTS' command where you define what you want. The end command '/LIST' and the area in between (the template area) where you define the html you want to see for each node.

The 'OBJECTS' part tells mmbase we want to list multiple nodes from one type

The TYPE="urls" tells mmbase we want a list of url nodes

The FIELDS="number,url,description" tells mmbase we want to use number, url and the description field of each url node.
