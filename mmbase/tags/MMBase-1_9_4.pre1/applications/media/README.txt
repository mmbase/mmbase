How to install.

0) Place mmbase-media.jar in your WEB-INF/lib

1) Copy also the WEB-INF/config/media directy. You need to add your own configuration there.

2) The needed builders and applications are present in mmbase-media.jar
   
   If you want to change them, you may place alternate versions in WEB-INF/config

3) The example application 'MediaEditor' depends on cloud context security and editwizards.
   Templates can be found in web-app/edit. Copy those to your web-app, e.g. to /mmbase/mediaedit.

4) Restart MMBase.
