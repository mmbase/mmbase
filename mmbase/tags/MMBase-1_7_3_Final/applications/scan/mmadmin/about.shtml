MMBase 1.5

Welcome to Runner the quick unzip & go version of the MMBase Content management system.

This is runtime version aimed at end users of mmbase to allow people to use MMBase within minutes of downloading the system and get going. This version is for people who are oke with the bundled server and database for the other setups or if you want to develop/change MMBase we advince you to use the MMDistro developers version also available on the www.mmbase.org site. That contains full sources and build tools.

If you run into problems you can get help from the following sources :

readme/releasenotes/installation docs - These documents are in the root of the binary distro
website - http://www.mmbase.org
mailing lists - http://www.mmbase.org/communication
irc channel - #mmbase on ircnet (for example irc.xs4all.nl)

The MMbase Release Team.

About Orion :

Orion is a great j2ee application server that is 100% java based, easy to install and follows all the new and hip specs. Its not free but at $1500 its very cheap indeed and it is free for development and non-profit use. Many thanks to the orion people for allowing us to include it in the demo version. More info on orion at www.orionserver.com.

About Hypersonic :

Hypersonic is a small 100% java based JDBC database, its opensource, fast very small (about 100kb). It also supports a memory only mode what we use in the demo meaning that its empty once you stop the server. For more info on Hypersonic at http://hsql.oron.ch.
MMBase Distro version

Welcome to MMBase Content management system.

The Binary distro includes a database. Since we wanted a simple and 100% java solution we used Hsqldb, but MMBase itself supports many more both commercial and opensource products.

The Source distro includes all the sources to build your own MMBase.
