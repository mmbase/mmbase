Welke gegevens kan ik bekijken in voortgang?

Voortgang bekijken

Klik op voortgang. U ziet in het voortgang scherm de volgende gegevens:

-        Percentage doorlopen

-        Werkgroep :

-        Aantal maal ingelogd

-        Duur inloggen

-        Naam opleiding

-        Informatie over toetsen bij de opleiding
