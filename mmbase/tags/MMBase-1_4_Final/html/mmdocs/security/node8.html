The configuration

In the configuration file security/security.xml there are the settings of the security. This file contains the following tags:
  • /security with the follwing attributes:
    • active This can be either true or false. When it is false a dummy security configuration will be loaded, otherwise the config file will be processed.
  • /security/authorization with the follwing attributes:
    • class : The class which should be used for authentication, must be an derived instance of class org.mmbase.security.Authorization.
    • url : The url of the configfile which should be used for configuration of of the authorization class.
  • sharedsecret, which should contain a value which is used by remoteBuilders to check if a another system is also trustable.
