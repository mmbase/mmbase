Status of Security

NOS Internet, Eduard Witteveen

The current status of the security system which is now in the cvs of MMBase.

Abstract:

The current status of the security system which is now in the Concurrent Versions System (CVS) of MMBase. This document will explain the steps which were made to get to this point and describes how the (currently 2) implementations of the security system functionate. The current CVS has now security in the bridge classes. This means the following:
  1. A cloud can be obtained by telling which authentication should be used and providing a hashmap with the properties
  2. Every operation on the cloud is checked (except the multilevel list)
  3. Every operation on the transactions is checked (except the multilevel list)
