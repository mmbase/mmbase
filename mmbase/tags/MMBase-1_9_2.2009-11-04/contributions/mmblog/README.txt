MMLog is a simple weblog system. It allows you to create a weblog, add items to it. Comes with a backend/admin tool and a frontend add/view part.

- You need a servlet 2.4 application server like Tomcat 5. 
- You need MMBase 1.8

To install MMBlog from bin
--------------------------------------------------------------------------------
  copy WEB-INF/lib/mmbase-mmblog.jar to your WEB-INF/lib
  install the webpages found in templates/jsp/mmblog in your webdir '/mmblog'

  restart your application server

  Once installed you can open the mmblog system by visiting mmblog/index.jsp


To compile MMBlog and install from sources.
--------------------------------------------------------------------------------
  build and copy build/mmbase-mmblog.jar to your WEB-INF/lib
  install the webpages found in templates/jsp/mmblog in your webdir '/mmblog'

  restart your application server

  Once installed you can open the mmblog system by visiting mmblog/index.jsp
